# cuz inbox

Raw, unfiltered knowledge captured automatically from your Claude Code sessions.
Append-only. High-volume, recall-biased — capture over-collects; distill culls.

Each batch is stamped `<!-- date · project · session · event -->`. Lines are
tagged `#procedure` / `#fact` / `#pattern` / `#decision`.

**This is the capture layer, not the library.** Run `/cuz-distill` to cluster
these into real skills/memory and clear what's noise. Everything below the first
`---` is captured entries (none yet).
